# Style Guide

## Modules or files naming

1. By default name files using hyphen, like this:

   * `foo-bar.js`
   * `some-name.md`
   * `some-other-name.svg`

2. If a name of a file/module is attached to a component, class or a function name then name it the
   same way, like this (easy for searching over the project and good for consistency):

   * `AllNiches/index.js`
   * `obtainPageData.js`
   * `getSubPage.js`
